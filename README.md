# AnshuPostCraft

Turn messy text into engaging LinkedIn posts.

## Features

- **Ghost-writer formatting** — Automatically breaks raw text into short, punchy, story-style lines that perform well on LinkedIn
- **Rich text toolbar** — Bold, italic, bold+italic, and highlight selected text directly in the editor
- **Quick inserts** — One-click arrow bullets, bullet points, line breaks, and emojis (💡 🔑 🚀)
- **Live LinkedIn preview** — See exactly how your post will look on LinkedIn before you copy it, complete with profile card, reactions, and action buttons
- **See more / expand** — Preview mimics LinkedIn's "…see more" behaviour for long posts
- **One-click copy** — Copies clean plain text (no markdown symbols) ready to paste into LinkedIn
- **Download post** — Save your formatted post as a `.txt` file
- **Auto branding** — Appends "Formatted using AnshuPostCraft" to every post
- **Get 100 Viral LinkedIn Hooks** — Email button to request a free hooks list directly from the creator
- **Improve Hook** — Coming soon: rewrites your opening line into something people can't scroll past
- **Fully responsive** — Works on mobile and desktop
